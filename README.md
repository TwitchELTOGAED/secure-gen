# secure-gen

Proyecto Personal. Generador de Contraseñas Seguras y Correos Temporales

---

## ¿Qué es secure-gen?

secure-gen es una herramienta web gratuita que te ayuda a crear contraseñas fuertes y únicas, y a generar direcciones de correo temporal para usar en registros online, pruebas y cualquier situación donde no quieras exponer tu correo real.

---

## ¿Qué puedes hacer aquí?

- **Generar Contraseñas Seguras:** Elige la longitud, tipos de caracteres y obtén claves robustas para proteger tus cuentas.
- **Crear Correos Temporales:** La web genera un email desechable y te permite ver la bandeja de entrada para recibir mensajes en tiempo real.

---

## ¿Para quién está hecho?

- Personas que buscan proteger su privacidad al registrarse en sitios web poco conocidos.
- Usuarios que quieren contraseñas difíciles de adivinar sin complicarse.
- Cualquiera que necesite una solución rápida sin descargar programas ni crear cuentas.

---

## ¿Cómo usarlo?

1. **Abre la web:** desde el enlace público o clonando el repositorio.
2. **Selecciona la opción:** Contraseña segura o Correo temporal.
3. **Personaliza las opciones:** (longitud, caracteres, etc.) y presiona el botón de generación.
4. **Copia y usa:** la contraseña o el correo donde lo necesites.

---

## Tecnologías utilizadas

- **HTML, CSS y JavaScript:** Todo el código se ejecuta en el navegador.
- **1secmail API:** Para gestionar los correos temporales y mostrar la bandeja de entrada.

---

## Aviso de privacidad

No se almacena ningún dato personal. Los correos temporales son gestionados por 1secmail solo durante un tiempo limitado para máxima privacidad.
